# Rogue Bachata

Official website for Rogue Bachata workshops by Oscar Castellino.

**Website:** [roguebachata.com](https://roguebachata.com)

## About

Rogue Bachata is a movement practice that brings opera stage presence to the dance floor, drawing from Contact Improvisation, Alexander Technique, and the connection principles of tango, zouk, and kizomba.
